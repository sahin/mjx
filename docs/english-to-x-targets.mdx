## Introduction

This document provides a comprehensive list of potential compilation targets for "English-to-{X}" systems, where natural language serves as the source code. It begins by highlighting the **5 core compilation targets** that form the foundation of the MJX project, each with a detailed architecture specification.

Following the core targets is a broader reference list of **100 potential targets** organized into 10 distinct categories, covering a wide range of domains from user interface development to infrastructure management. This list is intended to serve as a source of inspiration and a reference for future research and development in the field of natural language-driven code generation.
